Распиновка портов ввода-вывода RPi Pico
==========================================

| Наименование порта | Обозначение порта RPI Pico | Примечание |
|:-------------------|:-----:|--------------------:|
| DIO 0 | GPIO 18 | Прерывания RFM95W |
| DIO 1  | GPIO 20 | Прерывания RFM95W |
| DIO 2  | GPIO 2 | Прерывания RFM95W |
| DIO 3  | ---- | Не используется |
| ---- | ---- | ---- |
| RST | GPIO 15 | 600 |
| MISO | GPIO 12 | 700 |
| MOSI | GPIO 11 | 800 |
| SCK | GPIO 10 | 900 |
| NSS | GPIO 3 | 1000 |
| BUSY | ---- | Не используется |
| RXEN | ---- | Не используется |
| TXEN | ---- | Не используется |
| ---- | ---- | Не используется |
| SDA | GPIO 4 | 1000 |
| SCK | GPIO 5 | 1000 |
| ---- | ---- | ---- |
| GPS_EN | ---- | Не используется |
| GPS_RX | GPIO 9 | вход приемника UART |
| GPS_TX | GPIO 8 | выход передатчика UART |
| ---- | ---- | ---- |
| BUTTON_PIN | GPIO 17 | Кнопка выбора режимов работы |
| BATTERY_PIN | GPIO 26 | Напряжение питания (резистивный делитель) |
| EXT_PWR_DETECT | ---- | Не используется |
| Monitor Pin | ---- | Не используется |
| LED_PIN | GPIO 25 | Светодиод |
| BEEP | ---- | Не используется |
